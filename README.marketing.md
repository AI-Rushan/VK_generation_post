# Генератор постов для товаров

Быстро превращает ссылку на товар в готовый продающий пост для ВКонтакте с помощью LLM.

## Почему это удобно

- Экономит время на подготовке контента для соцсетей.
- Помогает выдерживать единый стиль и тон публикаций.
- Дает гибкую настройку текста под задачу (тон, длина, CTA, эмодзи, хештеги).
- Позволяет сразу публиковать или планировать посты в VK.

## Что умеет

- Генерация поста по ссылке на товар.
- Быстрое редактирование и копирование результата.
- Публикация в сообщество ВКонтакте.
- Отложенные публикации.
- Избранное для сохранения лучших вариантов.

## Для кого

- Владельцы небольших магазинов.
- SMM-специалисты.
- Маркетологи, которым нужен быстрый контент-пайплайн.

## Ключевые технологии

- Flask
- ProxyAPI (OpenAI-совместимый API)
- VK API
- BeautifulSoup4

## Быстрый старт

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python app.py
```

После запуска откройте `http://127.0.0.1:5500`.
