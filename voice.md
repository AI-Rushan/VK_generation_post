# Настройки стиля постов

## Общий тон
- Позитивный и понятный.
- Общение на языке пользы для покупателя.

## Длина поста
- По умолчанию: средняя (3-5 смысловых блоков).
- Короткий: для быстрых публикаций.
- Подробный: когда нужно раскрыть преимущества.

## Количество эмодзи
- 1-4 эмодзи на пост.
- Эмодзи поддерживают акценты, но не заменяют текст.

## Стиль заголовков
- Короткий, конкретный, с названием товара.
- Заголовок должен сразу объяснять ценность.

## Стиль CTA (призыв к действию)
- Мягкий: приглашение написать и уточнить детали.
- Прямой: явный призыв к покупке.
- Срочный: ограничение по времени или количеству.

## Правила для хештегов
- 3-4 тематических хештега.
- Без слишком общих и нерелевантных тегов.
- Теги должны отражать формат предложения (акция, бренд, выгода).

## Чего избегать
- Слишком длинных и перегруженных фраз.
- Обещаний, которые нельзя подтвердить.
- Избыточных эмодзи и капслока.
